# Alumini - Frontend (Vite + React + Tailwind)

Install & run:
```
cd frontend
npm install
npm run dev
```
The dev server uses port 5173 by default.

Default admin credentials (backend):
- admin@college.com / admin123

Notes:
- This frontend expects the backend to run on http://localhost:5000
- Tailwind is configured; if you see unstyled UI, ensure you installed dev dependencies and run the dev server with Vite.
